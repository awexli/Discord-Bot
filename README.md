# Discord-Bot  
A discord bot that can take in various commands. Used for my personal server to moderate, but mostly to entertain.
## Commands  
- p-help  
- p-server   
- p-prune  
- p-avatar  
- p-ping   
- p-vid  
>>p-vid haiku  
>>p-vid song  
- p-doggo
